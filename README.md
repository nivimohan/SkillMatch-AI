# 🚀 SkillMatch AI

SkillMatch AI is an AI-powered resume analysis platform that helps job seekers evaluate their resumes, identify skill gaps, improve ATS compatibility, and receive personalized recommendations based on target job roles.

## 🌐 Live Demo

[View Live Application](https://skillmatch-aii.lovable.app)

---

## ✨ Features

### 📄 Resume Analysis
- Upload resume files
- AI-powered resume evaluation
- Resume scoring system
- ATS compatibility analysis

### 🎯 Skill Gap Detection
- Compare resume skills against target roles
- Identify missing skills
- Personalized improvement suggestions

### 👤 User Authentication
- Email and password login
- Google Sign-In
- Secure authentication with Supabase

### 📊 Smart Insights
- Resume strengths and weaknesses
- Improvement recommendations
- Career guidance suggestions

### 🎨 Modern UI
- Responsive design
- Clean and professional interface
- Mobile-friendly experience

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/SkillMatch-AI.git
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build Project

```bash
npm run build
```
